October 8, 2009: Several changes were implemented to improve and streamline the user experience. Some of the new features include:

  • Creating rotations in any division order.
  • Expanded main menu for easier navigation and more intuitive interaction.
  • Disabled the weather reporting feature due to lack of interest. Race conditions are no longer stored for each race.
  • More prominent feedback/announcement mechanism.
  • Short FAQ on creating rotations.

Still working on solving Javascript issues related to using Internet Explorer when entering finishes.

TechScore is a new regatta scoring system for CollegeSailing.org.

It extends the traditional scoring methods to bring all the data from all the collegiate regattas into one central database, allowing in the process real-time around-the-world update of ongoing regattas as well as past regattas. More information soon! This program is still under development.
